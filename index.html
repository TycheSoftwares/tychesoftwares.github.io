We <3 Open Source.

At Tyche Softwares, we have a special place in our hearts for open source.

We develop premium WooCommerce plugins that are accompanied with independent free WooCommerce plugins as well, or sometimes freemium plugins. Some of those open source plugins are:

  • Abandoned Cart Lite for WooCommerce

    A plugin that helps you recover abandoned carts from your WooCommerce shop by sending email notifications to customers.

  • WooCommerce Print Invoice & Delivery Notes

    This plugin allows you to print out invoices and delivery notes for your WooCommerce orders. You can also add personal notes, company policy, footer, etc.

  • Order Delivery Date for WooCommerce

    It helps you to improve your customer service by delivering the orders on the customerâ€™s specified date & time. Customers choose their order delivery date & time on the WooCommerce checkout page.

  • Product Delivery Date for WooCommerce

    If you are looking to capture different delivery dates for your WooCommerce products, this is the plugin you will need.

  • Export WordPress Menus

    This plugin allows you to export your WordPress Menus between different WordPress websites.

  • Prevent Customers To Cancel WooCommerce Orders

    This plugin restricts customers from cancelling a WooCommerce order. It will hide the Cancel button on My Account page for all user roles, except administrator.

How do I contribute?

We encourage contributions on all of the projects listed above. Please be sure to read the CONTRIBUTING.md file in each project (if one is present) prior to contributing your issue or pull request.

If you're technically inclined and would like to submit a pull request, we welcome it! Prior to submitting your pull request, we'd also advise reading the Github Help pages, which explain everything you need to know about setting up and working with Git and Github.

We're looking forward to your contributions!
